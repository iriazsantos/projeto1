
### Features por Aba

**📊 Dashboard**
- Status do gateway (conectado/desconectado)
- Resumo financeiro (total, pendentes, qtd)
- Últimos 5 pagamentos

**⚙️ Configuração**
- Seletor de provider (Asaas, Mercado Pago, Stripe)
- Input de API Key (mascara)
- Testar conexão ✓
- Salvar configuração

**➕ Nova Cobrança**
- Inputs: nome, email, CPF, valor
- Método: PIX | Boleto | Cartão
- Gera: PIX Code + QR Code
- Persiste no banco

**📜 Histórico**
- Filtros: Todos | Pendentes | Pagos | Falhas
- Listagem com scroll
- Status visual colorido
